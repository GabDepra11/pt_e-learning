Formatting Layout & Selectors

Mastering Display, Containers, and Interaction States

I. The Display Property

The display property dictates how elements are positioned relative to each other.

Keyword Description
inline Horizontally laid out.
block Vertically laid out.
list-item List item <li>.
none No box or no effect.

II. Structural Elements: Div vs. Span

<div> (Block-level): Used to organize larger sections.

Notice how I (the <p> tag) start on a new line!

<span> (Inline-level): Used for small parts. For example, this specific phrase is styled using a span without breaking the paragraph!

III. CSS Selectors (Class & ID)

Selectors are used to target elements for styling.

Class Selector (.)

Applies to a group of elements.

ID Selector (#)

Applies to exactly ONE element.
