CSS Formatting Layout

Understanding Box Types, Containers, and Selectors

1. The Display Property

The display property is the most important tool for controlling layout. It determines how the "box" of an element is rendered.

Keyword Description
inline Horizontally laid out.
block Vertically laid out.
list-item List item <li>.
none No box or no effect (removes from layout).

Kinds of Elements

Block-level:

I am a <p> tag. I take up the full width!

I start on a new line.

Inline-level:
I am an &em; tag. We stay horizontal.

2. Structural Containers: Div vs Span

Div Tag (<div>): A block-level element used to organize large sections. It always starts on a new line with space before and after it.
Span Tag (<span>): An inline element used to structure and emphasize specific words within a paragraph.

3. CSS Selectors (Class & ID)

Selectors point to the HTML elements you want to style.

Class Selector (.)

Applies the same style to a group of elements. Use a dot in CSS.

.Style1 { color: blue; }

ID Selector (#)

Applies a unique style to exactly one element. Use a hash in CSS.

#header1 { font-size: 20px; }
