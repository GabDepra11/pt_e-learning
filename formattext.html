Styling Text & Layout in CSS

A Comprehensive Learning Module for Web Developers

A. Font Properties (The Character Shape)

Styling text in CSS refers to how text is arranged inside its container. Text behaves like a series of inline elements, flowing naturally until it hits the edge of the content box.

Property Definition Values
color Specifies text color Red, #ff0000, rgb(255,0,0)
font-family Indicates the typeface Arial, "Times New Roman", sans-serif
font-size Indicates the size px, em, %, small, large

B. Text Layout Styles

Note on Text-Align: The text-align property can be set to left, center, right, or justify. Justify stretches text so each line has equal width, common in formal publications.

C. The Box Model: Borders & Spacing

Groove Border
border-style: groove;
Rounded Border
border-radius: 20px;

Margin vs. Padding: Padding is the space inside the element (between content and border). Margin is the space outside the element (between neighbors).

D. Lists & Dynamic States

  • List-style-type: disc, circle, square, or roman numerals.
  • Pseudo-classes: Styles based on conditions (e.g., :hover).
  • Pseudo-elements: Styles specific parts (e.g., ::first-letter for drop caps).

D rop caps can be created using the ::first-letter pseudo-element. This creates a visually appealing start to a paragraph, often used in magazine layouts.
