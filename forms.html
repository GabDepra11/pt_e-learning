What is a Webform?

A form is a window or screen that contains numerous fields, or spaces to enter data. It acts as a container for different types of input elements such as text fields, checkboxes, and buttons.

Core Form Attributes

Action: What happens when submitted.
Target: Where the result will open.
Method: How data is sent (Get or Post).

Attributes of the <input> Tag

Attribute Name Values
name Any text (without spaces)
type Text, password, radio, checkbox, etc.
size Any number (width of input)
checked No value (pre-selects radio/checkbox)
