Understanding HTML Frames

Partitioning Web Windows & Embedding Documents

1. Legacy Framesets

Frames: A partition in a window that allows you to display multiple web pages. You can create frames by using the <frameset> tag.

Frameset Elements: Divides the browser window into separate frames using cols and rows attributes. These take percentage values separated by commas.

  • Cols: Indicates the number and width of columns.
  • Rows: Indicates the number, height, and rows.

2. Frame & Iframe Attributes

Both standard frames and inline frames use these specific attributes to control behavior and appearance:

Attribute Description
frameborder Turns the frame border on and off (0 or 1).
noresize Locks the border to prohibit user resizing.
name Defines the name used as a target for links.
src Indicates the web page or file to be displayed.

3. Inline Frames <iframe>

Unlike standard frames, Inline Frames are embedded directly within the <body> of a webpage.

  • Used to embed another document within the current HTML document.
  • They do not require a separate frameset file.
  • Essential for embedding maps, videos, or external widgets.
