Introduction to CSS

Formatting the Layout of the Web

What is CSS?

CSS (Cascading Style Sheets) is a language used for specifying the style of documents presented to users. It is primarily used to format the layout of Web pages.

Stylesheet: Refers to the document itself.
Cascade: A web style sheet is made to work together with other sheets to control the final look.

CSS Syntax & Structure

To style an element, you must follow a specific syntax:

h1 {
  color: blue; /* Property and Value separated by a colon */
  font-size: 12px; /* Declaration ending with a semi-colon */
}
  • Selector: Points to the HTML element you want to style.
  • Curly Braces { }: Surrounds the declaration block.
  • Colon (:): Separates the property name from the value.
  • Semi-colon (;): Signifies the end of a declaration block.

Three Ways to Apply CSS

Type Location Best For...
Inline CSS Inside the HTML element attribute. Styling a single specific element.
Internal / Embedded Inside the <style> tag in the head. Unique style for a single HTML page.
External CSS Separate .css file linked via <link>. Styling many web pages at once.
